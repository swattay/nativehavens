Login Help

In order to use this application, you will need to have cookies and javascript enabled on your Internet browser. It is also a good idea to set your page cache settings as described below.

  •  TURNING ON COOKIES (SESSIONS):

    Because the application operates via session identifications, you must configure your browser so that it allows such information to be passed to the SMT. On Internet Explorer's top menu, click on Tools>Internet Options. Under the "Privacy" tab, move the slider to the left until your settings are at or below "Medium."

  •  ENABLING JAVASCRIPT:

    One of the following browsers is recomended: Internet Explorer 6.0 or higher, Mozilla Firefox 2.0 or higher, Safari 2.0 or higher. Javascript must be enabled. If you have manually disabled javascript, you must re-enable it to use the application.
  •  INTERNET CACHE SETTINGS:

    For the application to function properly, you must allow temporary files and other information to be stored in you internet cache, or "Temporary Internet Files." If you are using Internet Explorer's, click on Tools > Internet Options. Under the "General" tab, click on the button labeled "Settings." Make sure that the appropriate option is selected so that you browser automatically checks for newer versions of stored pages.
